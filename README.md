# AWS Ticket Manager
A Python project to generate and manage AWS support tickets.

## Features
* Generate a pre-filled AWS support ticket with all required artifacts
* Preview the ticket before sending
* Submit the ticket and record the ticket ID
* Display the ticket ID and status in the dashboard and update via polling

## Requirements
* Python 3.8 or later

## Installation
* Install the project using pip: `pip install .`

## Usage
* Run the project using: `python -m aws_ticket_manager`
