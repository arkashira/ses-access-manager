# SES Access Manager

A simple Python project to manage SES access and track metrics.

## Usage

1. Create a new instance of the `SESAccessManager` class.
2. Record access granted for a customer using the `record_access_granted` method.
3. Get metrics using the `get_metrics` method.
4. Export metrics to the analytics DB using the `export_to_analytics_db` method.
5. Update the admin dashboard using the `update_admin_dashboard` method.
